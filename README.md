# michaels-world

Claude Code plugin marketplace.

## Plugins

### tmux-titles

Smart tmux window and pane titles for Claude Code sessions.

- Per-pane status icons in pane borders
- AI-generated descriptive window names via Claude Haiku
- Multi-pane awareness — combines context from all panes into one short title
- Priority-based bubble-up — window shows the highest-priority status across panes

## Installation

```bash
claude plugin marketplace add mshuffett/michaels-world
claude plugin install tmux-titles@michaels-world
```
